# 📊 HR Analytics Report | Excel, SQL & Power BI

An end-to-end HR Analytics project focused on employee attrition analysis and workforce insights. The project includes data cleaning in Excel, SQL-based exploratory analysis, and an interactive Power BI dashboard to identify employee turnover trends and key HR metrics.

---

## 🚀 Project Overview

This project analyzes employee data to understand workforce demographics, attrition patterns, job satisfaction, salary distribution, and employee experience.

The objective is to help HR teams identify factors contributing to employee turnover and support data-driven decision-making.

---

## 🛠️ Tools Used

- Microsoft Excel – Data Cleaning
- MySQL – Data Analysis
- Power BI – Dashboard Development
- GitHub – Project Documentation

---

## 📂 Project Workflow

### Data Cleaning
- Removed inconsistencies and duplicate records
- Validated employee records
- Standardized categorical fields
- Prepared clean dataset for analysis

### SQL Analysis
Performed exploratory analysis using SQL, including:

- Employee Count Analysis
- Attrition Analysis
- Department-wise Analysis
- Salary Analysis
- Overtime Impact Analysis
- Job Role Analysis
- Employee Retention Metrics

### Dashboard Development
Created an interactive Power BI dashboard to monitor workforce and attrition metrics.

---

## 📊 Dashboard Features

### KPI Cards
- Total Employees
- Total Attrition
- Attrition Rate (%)
- Average Age
- Average Years at Company

### Visualizations

#### Employee Attrition by Department
Shows departments contributing most to employee turnover.

#### Attrition by Salary Slab
Analyzes attrition trends across different salary ranges.

#### Job Satisfaction by Job Role
Displays employee satisfaction levels across various job roles.

#### Employees by Age Group
Shows workforce distribution among age categories.

#### Attrition by Gender
Compares employee turnover between male and female employees.

#### Attrition by Total Experience
Examines how employee experience influences attrition.

#### Department-wise Employee Distribution
Visualizes employee count across departments.

### Interactive Filters
- Department Filter
- Age Group Filter

---

## 📈 Key Insights

- Attrition varies significantly across departments.
- Salary levels appear to influence employee retention.
- Certain job roles show lower satisfaction levels.
- Workforce is concentrated within specific age groups.
- Employee experience impacts attrition behavior.
- Gender-based attrition patterns can be observed.
